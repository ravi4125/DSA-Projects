# Dynamic Memory Allocation using Linked List (C++)

## Project Overview
This project is a simple implementation of a Singly Linked List using Dynamic Memory Allocation in C++. It demonstrates how memory is allocated and deallocated using `new` and `delete`. The program is menu-driven and performs basic CRUD operations on the linked list.

## Features
- Insert Node at Beginning
- Append Node at End
- Display Linked List
- Search Node
- Delete Node
- Reverse Linked List

## Concepts Used
- C++
- Classes and Objects
- Constructor
- Dynamic Memory Allocation
- Linked List
- Pointers

## How to Run
1. Compile the `main.cpp` file.
2. Run the program.
3. Select the required operation from the menu.

## Conclusion
This project helps in understanding the basics of dynamic memory allocation, linked lists, pointers, and CRUD operations in C++.
